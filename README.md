# Open Calendar Toronto

This is a project to collect events from Toronto event venues and organizers and aggregate them into a single feed.

To see the latest data from this project view the (newest-data)[https://github.com/CivicTechTO/toe/tree/newest-data] branch of this project.

## Contributing

This project is meant to be easy to contribute to, cheap to run, and resilient over time. There are a couple ways to contribute to the project.

- (Submit a new data source)[https://github.com/CivicTechTO/toe/issues]
- (Code a data scraper)[https://github.com/CivicTechTO/toe/blob/newest-data/CONTRIBUTING.md]

If you would like to contribute to the overall structure of the project or the infrastructure the best method is to start in the `#proj-toronto-events` channel inside of Civic Tech Slack channel.
